# Zodsidian

Zod-based YAML frontmatter schema enforcement for Obsidian vaults.

## Packages

- **`@zodsidian/core`** — Parsing, validation, autofix, indexing
- **`@zodsidian/cli`** — CLI for validate, fix, index, report
- **`@zodsidian/obsidian-plugin`** — Obsidian plugin with live validation
- **`@zodsidian/schemas`** — Zod schema definitions

## Development

```bash
pnpm install
pnpm build
pnpm test
```
